objetivo general

Aprender las bases para el diseño de sitios web usando control de versiones como Git & GitHub.

objetivo general

  • Aprender las bases del uso de Git & GitHub para el control de versiones.
  • Aprender a usar los comandos básicos de Git usnado la consola.
  • Aprender como se estructura HTML5.
  • Aprender lostag que distribuyen el contenido en HTML.
  • Aprender a aplicar estilos usando CSS básico.
  • Aprender a crear nuestros propios estilos y usando CSS3.
