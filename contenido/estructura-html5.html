Estructura HTML5

Para aprender sobre la estructura de HTML5 de sus etiquetas y más, vamos a conocer un poco sobre ¿Qué es HTML?,¿Qué es HTML5? y ¿Cuáles son sus etiquetas o tags?

¿Qué es HTML?

HTML (HyperText Markup Language) es el bloque de construcción más básico de la Web. Define el significado y la estructura del contenido web. Tomado de MDN Mozilla

¿Qué es HTML5?

HTML5 es la última versión de HTML. El término representa dos conceptos diferentes:

  • Se trata de una nueva versión de HTML, con nuevos elementos, atributos y comportamientos.
  • Contiene un conjunto más amplio de tecnologías que permite a los sitios Web y a las aplicaciones ser más diversas y de gran alcance. A este conjunto se le llama HTML5 y amigos, a menudo reducido a HTML5 .
Tomado de MDN Mozilla

¿Cuáles son sus etiquetas o tags?

Los tags o etiquetas son elementos que le dicen al navegador como debe formatear el texto para poder visualizarlo, teniendo en cuenta que en HTML5 se incluyeron nuevas etiquetas, puede conocerlas todas aquí.

Estructura Tags HTML5

Las etiquetas cuentan con una estructura similar en todas, pero pueden diferir en algunos atributos. Existen dos tipos de etiquetas, las que tiene apertura - cierre y las que solo tienen apertura.

Un ejemplo de etiqueta de apertura - cierre sería:
<p>Texto</p>
Donde podemos observar que tanto <p> como </p> difieren en el /, el cual indica que es la etiqueta de cierre para el tag p de párrafos.

Un ejemplo de etiqueta de sólo apertura sería:
<img>
Donde podemos observar que no existe </img> para indicar el cierre, pues esta etiqueta no la necesita para poder cargar una imagen en nuestra página web.

Atributos y Valores

Todas las etiquetas tienen atributos y cada uno permite valores, para poder cargar texto, imágenes, videos, dar estilos y mucho más.

Teniendo en cuenta esto, todas las etiquetas mantienen la misma estructura para formarse y poder cargar valores en sus atributos, de la siguiente manera:

Etiqueta de sólo apertura: <etiqueta atributo="valor">
Ejemplo: <img src="assets/img/mockup.png">

Etiqueta de apertura - cierre: <etiqueta atributo="valor">contenido</etiqueta>
Ejemplo: <a href="https://www.google.com/">Ir a Google</a>
