¡Bienvenido!

En este OVA aprenderás las bases para comenzar a diseñar tu propia web, teniendo conocimientos en los conceptos sobre HTML, HTML5, CSS y CSS3. Todo esto apoyado de muchas herramientas como Visual Studio, Git y GitHub. Esté último, Git & GitHub, te ayudara a llevar un control de versiones en tus archivos y tu código, permitiendo que puedas tener a mas personas colaborandote en el desarrollo de sitios web sin la necesidad de estar pasandote archivos comprimidos por correo. Además te daremos las herramientas necesarias para que puedas hacer maravillas y codifiques con mucho entusiasmo. ¡Te deseamos el mejor éxito y esperamos que aprendas mucho!
